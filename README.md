# Calendar-App-Vue

## Short Description

A Calendar App in VueJS with Vuex.

Future implementation: NodeJS || LocalStorage

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
